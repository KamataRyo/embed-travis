## Contributing

```
$ git clone https://github.com/KamataRyo/oembed-travis
$ cd oembed-travis
$ bash bin/install-wp-tests.sh wordpress_test root '' localhost $WP_VERSION
$ phpunit
$ npm install
$ npm test
```
